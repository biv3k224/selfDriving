#JavaScript Self driving Car
A simple Python project that uses OpenCV and NumPy to detect lane lines for a basic self-driving car simulation.

##Tech Used
- Python 3.x
- OpenCV  
- NumPy  
- Matplotlib (optional for visualization)

## Installation
```sh
git clone https://github.com/biv3k224/selfDriving.git
cd selfDriving
pip3 install -r requirements.txt

Features

Lane detection

Edge detection (Canny)

Hough Line Transform

Real-time or image-based processing
